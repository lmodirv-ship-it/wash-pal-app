
## حجم منصة H&Lavage الكاملة

### 📊 الأرقام الإجمالية

| المكوّن | الحجم | التفاصيل |
|---------|--------|----------|
| **الكود المصدري** (`src/`) | ~2.0 MB | 110 ملف، 13,801 سطر |
| **الأصول العامة** (`public/`) | ~515 KB | شعار + صور |
| **فيديو ترويجي** | ~11 MB | `carwash-hero.mp4` |
| **التبعيات** (`node_modules`) | ~380 MB | مكتبات التطوير فقط |
| **قاعدة البيانات** | < 5 MB | بيانات تجريبية |
| **Bundle بعد البناء** | ~800KB - 1.2MB | مضغوط Gzip |

### 🏗️ بنية المنصة

- **28 صفحة** (Landing, Login, Dashboard, Orders, Customers, Employees, Services, Invoices, Reports, Branches, Shops, Finance, Settings, Team, Admin pages, …)
- **~60 مكوّن** (shadcn/ui + مخصصة)
- **3 لغات** (عربي، فرنسي، إنجليزي)
- **19 جدول** في قاعدة البيانات
- **18 دالة PostgreSQL**
- **1 Edge Function** (`admin-camera-auth`)
- **30+ migration**

### 🌐 الحجم الفعلي للمستخدم النهائي

ما يُحمّله المستخدم عند زيارة الموقع:
- **التحميل الأولي**: ~1.2 MB (HTML + JS + CSS مضغوط)
- **مع الفيديو**: ~12 MB (إن شُغّل)
- **إجمالي الموقع المنشور**: ~15 MB

### 💾 ما لا يُحسب على المستخدم

- `node_modules` (380 MB) — موجود فقط في بيئة التطوير، لا يُرسل للمتصفح
- Migrations وملفات الإعداد — لا تصل للمستخدم

### 📈 الخلاصة

منصة **H&Lavage** هي تطبيق متوسط الحجم نسبياً:
- خفيف على المستخدم (~1-2 MB تحميل أولي)
- شامل من ناحية الميزات (28 صفحة + 19 جدول)
- الفيديو الترويجي (11 MB) هو أكبر عبء ويمكن تحسينه
