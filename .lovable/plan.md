

# برنامج "لافاج" - نظام إدارة محلات غسل السيارات

## نظرة عامة
تطبيق ويب شامل لإدارة محلات غسل السيارات يشمل جميع الميزات الأساسية.

## الميزات المطلوبة

1. **إدارة الطلبات والحجوزات** - إضافة طلب جديد، تتبع الحالة (انتظار، جاري، مكتمل)
2. **إدارة العملاء** - قاعدة بيانات العملاء، سجل الزيارات
3. **إدارة الموظفين** - قائمة الموظفين، تعيين المهام
4. **خدمات الغسيل** - غسيل خارجي، داخلي وخارجي، تلميع، غسيل محرك
5. **الفواتير والدفع** - إنشاء فواتير، طباعة، تسجيل المبالغ
6. **التقارير المالية** - إيرادات يومية/شهرية، إحصائيات
7. **دعم أكثر من فرع**

## الهيكل التقني

### الصفحات
- `/` - لوحة التحكم الرئيسية (Dashboard)
- `/orders` - إدارة الطلبات
- `/customers` - إدارة العملاء
- `/employees` - إدارة الموظفين
- `/services` - إدارة الخدمات
- `/invoices` - الفواتير
- `/reports` - التقارير
- `/branches` - إدارة الفروع
- `/settings` - الإعدادات

### المكونات الرئيسية
- `Sidebar` - قائمة جانبية للتنقل مع شعار "لافاج"
- `DashboardStats` - إحصائيات سريعة (طلبات اليوم، الإيرادات، العملاء)
- `OrderForm` - نموذج إضافة طلب جديد
- `OrdersTable` - جدول الطلبات مع فلترة وبحث
- `CustomerCard` - بطاقة معلومات العميل
- `InvoicePreview` - معاينة الفاتورة للطباعة
- `RevenueChart` - رسم بياني للإيرادات

### إدارة البيانات
- استخدام `localStorage` لتخزين البيانات مبدئيا (يمكن ربطه بـ Supabase لاحقا)
- React Context للحالة العامة (الفرع الحالي، المستخدم)
- React Query لإدارة البيانات

### التصميم
- واجهة عربية (RTL) بالكامل
- ألوان احترافية: أزرق داكن مع لمسات خضراء
- تصميم متجاوب يعمل على الكمبيوتر والجوال
- أيقونات Lucide React

## خطوات التنفيذ

1. **إعداد البنية الأساسية** - تفعيل RTL، إعداد الألوان، إنشاء Sidebar والتنقل
2. **لوحة التحكم** - إحصائيات، آخر الطلبات، رسم بياني
3. **نظام الطلبات** - إضافة/تعديل/حذف طلبات، تغيير الحالة
4. **إدارة العملاء والموظفين** - CRUD كامل
5. **الخدمات والأسعار** - قائمة الخدمات مع الأسعار
6. **الفواتير** - إنشاء وطباعة الفواتير
7. **التقارير** - رسوم بيانية وإحصائيات مالية
8. **إدارة الفروع** - إضافة فروع والتبديل بينها
